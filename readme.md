# Shadowrocket Modules

整理一些自己用到的 Shadowrocket 模块。

# 使用方法

复制 `.module` 链接，然后打开 Shadowrocket 软件的 **配置** tab，找到 **模块** 进入，之后点击右上角的 **+** 将复制的链接粘贴进去点**下载**即可。
